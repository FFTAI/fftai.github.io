Fourier-GRX SDK

概述

Fourier-GRX SDK 是一个软件开发工具包，提供了一套API用于控制和监控傅里叶智能的通用机器人产品。该SDK设计简洁易用，提供高级接口来控制机器人运动并读取传感器数据。SDK支持Python语言。

使用Fourier-GRX库

快速入门 是开始使用Fourier-GRX SDK库的推荐方式。这份分步指南将帮助您安装所需库文件并运行简单程序来控制机器人。

示例代码 演示了SDK库的使用方法，可作为您开发应用程序的参考程序。

参考指南 详细记录了SDK所有模块、类和接口的文档说明。

支持平台与版本

Fourier-GRX SDK库支持以下平台：

  • Ubuntu 22.04 LTS

支持以下Python版本：

  • Python 3.11
